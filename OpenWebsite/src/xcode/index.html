Mobile Application Development

*************iOS**************

The project Open website is a 2 screen mobile application. This application has been developed using storyboard in ios.

ViewController is the first screen. ViewController.h and ViewController.m - Controls all the actions that are performed on the first screen. This screen has a button and a text field. The url entered on the first screen is opened on the second screen. On click of 'Go' button, the user is taken to the second screen which is Web Screen.

WebScreen is the second screen. WebScreen.h and WebScreen.m - Controls all the actions that are performed on the second screen. This screen has a navigation bar with a bar button called 'Back' which takes the user back to the first screen. UIWebView is used to display the web page on the app.

Navigation : view controllers identifiers are used to identify different view controllers on the storyboard. Navigation view controller is used to control the flow.

Do not forget to enable user interaction by clicking checkbox for label
